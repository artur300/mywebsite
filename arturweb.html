Tetris Game

OOP and WinForms Implementation


Experience the classic Tetris game in a modern, object-oriented programming (OOP). This implementation combines the power of OOP principles and Windows Forms to bring the beloved Tetris game to life.




In this project, I've created a dynamic and engaging Tetris experience where players can control falling shapes, strategically placing them to complete rows and earn points. I've designed a user-friendly interface with player registration for different game modes, including Credit Card Payment, PayPal Payment, and Free-to-Play options.


Key Project Features:




OOP Design: Explore the elegance of OOP with well-structured classes for players, payment methods, and Tetris shapes.

Registration System: Choose your player type and register with ease.

Gameplay: Move and rotate Tetris shapes, clear rows, and increase your score.

Game Over Handling: Challenge yourself to achieve a high score without hitting game over.


This Tetris Game project showcases the fusion of OOP design and interactive gameplay.







Ecommrance Website

HTML,CSS and JAVASCRIPT Implementation


This project is a hands-on exploration of web development, showcasing my skills in HTML, CSS, and JavaScript. Dive into an immersive e-commerce experience designed for both learning and practical use.




This project is not just a website; it's a reflection of my commitment to continuous learning. Every line of code represents a step in my journey to mastering web development, and I'm excited to share the result of my efforts with you.


Key Project Features:




Smart Search Engine: Utilize the search bar for efficient item discovery based on color, size, and bag type. The interconnected system ensures a fluid shopping experience.


Dynamic Product Display and Cart Management: real-time updates on the delivery date using Days.js. The site seamlessly calculates taxes, and you can add or delete items within the cart, with changes reflecting across the entire platform.


Comprehensive Checkout Process: Navigate to the buy option even with an empty cart. In the payment section, choose your country to reveal a list of cities, enhancing the precision of your shipping information.


Automated Email Confirmation: Upon purchase completion, receive a detailed email with all your selected items, providing a comprehensive summary of your transaction.
